# Week 02 Assignments

This folder contains the assignments and tasks for Week 02 of the Data Engineering Bootcamp, focusing on data cleaning and analysis.

## Assignments Overview

### 1. [week02-assignment-1](./week02-assignment-1) - Article Similarity
This assignment involves calculating similarity between articles.
- Utilizes Python and Numpy for efficient calculations.
- Includes a Jupyter Notebook (`article_similarity_solution.ipynb`) with the implementation.
- Uses `articles.csv` as the primary dataset.

### 2. [week02-assignment-2](./week02-assignment-2) - Data Science Jobs Cleaning
Focused on data cleaning and preprocessing of a "Data Science Jobs" dataset.
- Includes `Uncleaned_DS_jobs.csv` and the corresponding `My_Cleaned_DS_Jobs.csv`.
- Implementation is provided in `Assignment.ipynb`.

### 3. [week02-assignment-3](./week02-assignment-3) - Cafe Sales Analysis
An assignment focused on cleaning and analyzing "Cafe Sales" data.
- Involves processing `dirty_cafe_sales.csv` to produce `Clean_Cafe_dataset.csv`.
- Tasks and requirements are detailed in `Cafe Sales Data Cleaning.docx`.
- Solution script is in `Assignment.ipynb`.
