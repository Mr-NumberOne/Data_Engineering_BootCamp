# Week 01 Assignments

This folder contains the assignments and tasks for Week 01 of the Data Engineering Bootcamp.

## Assignments Overview

### 1. [week01-project-1](./week01-project-1)
A Python-based analytics project focused on modular code organization. It includes:
- `main.py`: The entry point for the application.
- `models.py` & `analytics.py`: Logic for data modeling and analytics.
- `utils.py`: Utility functions.
- `data.csv`: Sample data for processing.

### 2. [week01-task-1](./week01-task-1)
Contains documentation (`python-tasks.docx`) outlining core Python programming tasks and exercises to reinforce foundational concepts.
